About Project

Vocabulary for class 9

HI USER!

Vocabulary for Class 9 is an open-source project created for 9th-grade students. The project contains a complete collection of English vocabulary from the school textbook, where each word is provided with: a Tajik translation English transcription (phonetics) clear organization by lessons (1–72) The main goal of this project is to make learning English easier and more accessible for students, teachers, and self-learners who want to study Grade 9 vocabulary in a simple and structured way. The project is completely free, ad-free, and does not require registration. Anyone can use the source code, improve it, or contribute to its development. Technologies used: HTML CSS JavaScript (Front-end only)

САЛОМ!

Vocabulary for Class 9 — лоиҳаи open-source аст, ки барои хонандагони синфи 9 таҳия шудааст. Лоиҳа луғатҳои пурраи китобҳои англисиро дар бар мегирад ва ҳар калима бо: тарҷума ба забони тоҷикӣ транскрипсияи англисӣ ҷудошуда аз рӯи lesson (1–72) пешниҳод мешавад. Ҳадафи асосии лоиҳа осон кардани омӯзиши забони англисӣ барои хонандагон, омӯзгорон ва ҳар касе мебошад, ки мехоҳад луғатҳои синфи 9-ро зуд ва фаҳмо азхуд кунад. Лоиҳа комилан ройгон, бе реклама ва бе бақайдгирӣ аст. Ҳар кас метавонад кодро истифода кунад, такмил диҳад ё саҳм гузорад. Технологияҳо: HTML CSS JavaScript (Front-end only)
